Lesson 1.2: Creating a Structured Document with HTML

Languages

The three most important languages in front-end developing are HTML, CSS and Javascript. HTML is like the structure of the house and the CSS is like the style of the house.

Elements

All elements on the page are rectangular and all text that is shown on the page can be found in the elements, deep down.

Tree structure of a page

Each element has an opening tag and a closing tag Inside a div-element can be a h2 element within there can be a span element containing text. Also within that div element can be a p element containing text and a elements

Boxes

Every element on the page is a box, although it can manifest itself in another form.

Lesson 1.3: Adding CSS Style and HTML Structure

What does cascading or sheets mean?

CSS will be referenced in HTML. The rules can be compared to the different levels of law: national law state law, municipality law, house rules.

Benefits of avoiding repetition

One example of the "national law" in CSS and HTML is this webpage is the div class called "lesson". Since all the boxes that cover a lesson are given this class-name, they can all be referred to the same tag in CSS (called .lesson).This way the styling specifications for all the different lesson boxes don't have to be written out multiple times. This allows the programmer to work faster and for changing the code faster later in the process. The same goes for the "state law" in this webpage, which is the div class called "concepts".
